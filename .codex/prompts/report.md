# Superpowers: report

Generate an engineering-ready handoff using the **report** skill.

- Branch/request to summarize: $ARGUMENTS
- Run: `~/.codex/superpowers/.codex/superpowers-codex use-skill superpowers:report`
- Include evidence (file paths, commands run, test results) and highlight follow-ups.

Deliver:
- What changed and why
- How to verify (tests/commands)
- Known gaps or risks + owners
