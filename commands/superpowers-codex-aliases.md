---
description: Codex slash-command aliases for loading superpowers skills
---

Use `/superpowers:<skill>` in Codex to load the matching superpowers skill. For example:

- `/superpowers:brainstorm` → loads `superpowers:brainstorming`
- `/superpowers:brainstorming` → loads `superpowers:brainstorming`

Normal `superpowers-codex` commands still work:

- `superpowers-codex find-skills`
- `superpowers-codex use-skill superpowers:<name>`
